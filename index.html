Desenvolvimento web

O que é desenvolvimento web?

No passado, o desenvolvimento web se resumia basicamente a criação de sites, mas hoje, ele é muito mais complexo e envolve muito mais elementos e linguagens. De modo geral, podemos dizer que desenvolvimento web é a área de Tecnologia que vai criar, codificar e programar páginas na internet. Lembrando que, os sites e demais programas presentes na internet hoje são muito mais dinâmicos e necessitam de um desenvolvimento maior, com diversas integrações com APIs (Application Programming Interface), preparação e armazenamento de banco de dados e muito mais. Essa complexidade atual e as várias linguagens utilizadas na programação acabou dando origem a profissões distintas como, por exemplo, a de front-end e back-end. É claro que ainda existem profissionais full que dominam ambas as linguagens de programação, mas dependendo do tamanho do projeto de desenvolvimento, esse profissional acaba sendo um gestor com a responsabilidade de integrar as áreas. Mais adiante vamos explicar melhor cada uma das profissões/funções dentro desse universo.

O que faz um profissional DEV?

O profissional de Dev, conhecido hoje como Web Developer, é o responsável por criar todo o planejamento estrutural de páginas, utilizando linguagens de programação como HTML, CSS, JavaScript, PHP, entre várias outras. E como falamos acima, a criação das páginas são cada vez mais complexas. Além de criar o site que será visível para o usuário, o profissional de Dev precisa criar toda a parte invisível, codificando todos os elementos presentes na página, considerando as integrações com APIs, armazenamento de dados, cibersegurança e usabilidade.
